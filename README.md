# Action Auctions

#### A web app built with Node.js on AWS

## Requirements
- Node.js
- Elastic Beanstalk CLI

## Setup

```
$ git clone https://github.com/freyaryd/ActionAuctions.git
$ cd ActionAuctions
$ npm install
```

## Running
```
$ npm start
```
